# soc-analyst-portfolio
I have a Master's in Applied Cyber Security and I am currently building the practical skills needed for a SOC Analyst role.

This repository documents my cybersecurity learning journey through Security+, TryHackMe, hands-on labs, and security investigations. The goal is to strengthen my understanding of core concepts while developing real-world blue team and SOC skills.

## Current Focus

* CompTIA Security+ (SY0-701)
* SOC Analyst fundamentals
* Threat detection and monitoring
* Incident response
* MITRE ATT&CK
* Splunk
* Microsoft Sentinel

## Repository Structure

### concepts/

Cybersecurity concepts explained in my own words with examples and key takeaways. These notes also support my Security+ preparation.

### thm-writeups/

Write-ups from TryHackMe rooms, including what I learned, tools used, and lessons gained from each exercise.

### home-soc-lab/

Projects and investigations from my home SOC lab, including detection, analysis, and incident response activities.

### case-studies/

Security incidents, attack scenarios, and investigation summaries used to improve analytical thinking and reporting skills.
