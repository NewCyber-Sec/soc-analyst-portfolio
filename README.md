# soc-analyst-portfolio

I have a Master's in Applied Cyber Security and I am currently building the practical skills needed for a SOC Analyst role.

This repository documents my cybersecurity learning journey through Security+, TryHackMe, hands-on labs, and security investigations. The goal is to strengthen my understanding of core concepts while developing real-world blue team and SOC skills.

## Current Focus
- CompTIA Security+ (SY0-701)
- SOC Analyst fundamentals
- Threat detection and monitoring
- Incident response
- MITRE ATT&CK, Splunk, Microsoft Sentinel

## Repository Structure
- **concepts/** — Cybersecurity concepts explained in my own words with examples and key takeaways. Also support my Security+ prep.
- **thm-writeups/** — Write-ups from TryHackMe rooms: what I learned, tools used, lessons gained.
- **home-soc-lab/** — Projects and investigations from my home SOC lab: detection, analysis, incident response.
- **case-studies/** — Security incidents, attack scenarios, and investigation summaries.
